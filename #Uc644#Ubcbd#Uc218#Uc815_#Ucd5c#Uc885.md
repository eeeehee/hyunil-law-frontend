## 💡 왜 이 문제가 발생했나?

### 원인 분석:
1. **폴더 구조 변경**
   ```
   이전: 모든 파일이 루트에
   이후: /pages/user/, /pages/admin/, /js/ 등으로 분리
   ```

2. **경로 일괄 변경 시 실수**
   - HTML 파일만 수정
   - **JavaScript 파일은 누락** ← 핵심 문제!

3. **JavaScript가 HTML을 동적 생성**
   - client_ui.js가 사용자 메뉴 생성
   - admin_sidebar.js가 관리자 메뉴 생성
   - 이 파일들의 오류로 메뉴가 잘못 생성됨

---

## 🔍 기술적 설명

### 동적 메뉴 생성 방식:
```javascript
// client_ui.js에서 사이드바 HTML을 생성
const sidebarHTML = `
    <nav class="sidebar">
        <ul class="menu-list">
            <li><a href="/pages/user/dashboard.html">대시보드</a></li>
            ...
        </ul>
    </nav>
`;
document.body.insertAdjacentHTML('afterbegin', sidebarHTML);
```

### 오류가 있을 때:
```javascript
// href="="로 인한 잘못된 HTML 생성
<a href="="/pages/user/dashboard.html">  // 작동 안 함!
```

### 수정 후:
```javascript
// 올바른 HTML 생성
<a href="/pages/user/dashboard.html">    // 정상 작동!
```
