# Hyunil Law - Restructured (corp)

This package reorganizes the original flat `corp` deploy into a standard folder layout and converts internal links/assets to **absolute paths**.

## Run locally (recommended)
From this folder, run:

```bash
npx http-server -p 5173
```

Then open:
- http://localhost:5173/

## Notes
- Pages are under `/pages/public`, `/pages/user`, `/pages/admin`
- Shared scripts are under `/js`
- Images are under `/assets/img`
